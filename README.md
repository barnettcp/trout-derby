# Trout Derby

A laid-back, arcade-style fishing game built in Godot 4. Players cast into a shared pond, compete to catch the most fish before the derby timer runs out, and chase rare trophy catches.

## Project Goals

This is a learning project with a few purposes:

- Practice game design documentation and planning workflows
- Learn Godot 4 and GDScript through a small, scoped project
- Explore simple multiplayer networking in a low-stakes context
- Build something that could realistically ship in a game jam weekend

The game itself is intentionally simple. Solid colors, minimal art, straightforward mechanics. The focus is on making the fishing loop feel good and the multiplayer work reliably, not on visual polish.

## Status

**Work in progress.** This project is in the design and documentation phase. Implementation may or may not follow. The docs in this repo represent practice in structured game design planning.

## Documentation

Design documents live in `docs/`:

- `game-design-document.md` - Overview and core loop
- `mechanics/` - Fishing, fish behavior, catalog, scoring
- `technical/` - Architecture, multiplayer, data models
- `art/` - Style guide, UI layout
- `milestones/` - Prototype and final game task lists
